Este es un proyecto sencillo, basado en la simpleza y la facilidad para el usuario de encontrar lo que está buscando,
en este caso comprar entradas, en la cabezera añadimos un título con el nombre del grupo y una foto del grupo, despues 
tenemos un breve descripción, una tabla con datos importantes para el usuario y por último el formulario a rellenar con
los datos necesarios para comprar la entrada. A través de javascript hemos mandado estos datos a otro página web llamada
Confirmación, para que el usuario confirme que todos los campos están correctamente. Esta sería la composición y función
de nuestra página.
Algunas de las fuentes referenciadas son "w3school", "Wikipedia" y compañeros que me han ido orientando en ciertas partes
del proceso.
